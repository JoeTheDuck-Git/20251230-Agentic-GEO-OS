# GEO Overview Page - Agent-to-UI Projection

## Page Purpose

The GEO Overview page is the main dashboard that provides a high-level view of brand performance in AI-generated answers. It aggregates metrics, insights, and recommendations from multiple agents to give users a comprehensive understanding of their GEO status.

## Intended Audience

- **Primary**: Brand managers, marketing leaders, content strategists
- **Secondary**: Executives, analysts, stakeholders
- **Use Case**: Quick status check, performance monitoring, strategic overview

## What This Page MUST NOT Allow

1. **No Score Editing**
   - Users cannot manually adjust GEO Scores
   - Scores are computed outputs, not editable

2. **No Recommendation Execution**
   - No "Execute" or "Apply" buttons
   - Recommendations are displayed for review only
   - Execution happens outside this system

3. **No Agent Triggering**
   - No buttons to trigger new analysis
   - No "Refresh" or "Recompute" buttons
   - Analysis is triggered through separate process

4. **No Scope Modification**
   - No UI to add/remove topics or brands
   - No scope expansion controls
   - Scope changes require separate process

5. **No Policy Changes**
   - No UI to modify policies or constraints
   - No governance rule editing
   - Policy changes require separate process

6. **No Data Export Without Approval**
   - No automatic data export
   - Export requires explicit user action and approval
   - Export is logged and audited
